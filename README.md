# rust_weather_app
First rust project!

A simple terminal based weather application to familiarize myself with the Rust programming language.

## Requirements
- [Rust](https://www.rust-lang.org/tools/install)

## Usage
    $ cd weather_app
    $ cargo run

Or using release build:

    $ cd weather_app
    $ cargo build --release
    $ ./target/release/weather_app
